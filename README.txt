Как обновлять новости (без бэкенда)
1) Откройте файл data/news.json.
2) Скопируйте и вставьте новый объект в начало массива.
3) Поля:
   - id: уникальная строка (только латиница/цифры/дефисы).
   - date: YYYY-MM-DD.
   - title: заголовок.
   - summary: краткое описание (1–2 предложения).
   - content: основной текст (поддерживается **жирный**, *курсив*, ссылки [текст](https://...)).
   - tags: ["метки", "через", "запятую"].
   - cover: путь к картинке (например, assets/photo.jpg), не обязательно.
4) Загрузите сайт на хостинг: GitHub Pages / Netlify / обычный хостинг.
5) Для картинок кладите файлы в папку assets и указывайте относительный путь.
